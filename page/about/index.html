关于

欢迎来到 highwe2hell.top！我是一个喜欢折腾各种工具和技术的编程爱好者，这个博客是我学习编程和探索技术的成果展示。在这里，我分享自己在代码编写、网站搭建以及日常电脑使用中的心得与经验，希望这些内容能为你带来帮助和启发。

为了打造和维护这个博客，我使用了一些得力工具。它们在各自的领域里都非常出色，助我一臂之力，现将这些工具一一介绍给大家。

网站生成与主题

  • Hugo：Hugo 是一款静态网站生成器，它的速度和灵活性使我能够快速构建和发布内容。通过简洁的配置文件和 Markdown 支持，Hugo 让我专注于内容创作，而无需过多关注网站的架构。
  • Stack：这是我在 Hugo 中选择的主题，它设计简洁，响应速度快，能够完美展现我的文章内容，同时提供了多种自定义选项，适合不同类型的博客需求。

代码管理与编辑

  • Git & GitHub：Git 是一个分布式版本控制系统，而 GitHub 则是一个基于 Git 的代码托管平台。我使用它们来管理和控制博客项目的版本，所有的博客源代码都托管在 GitHub 上。
  • VSCode：Visual Studio Code 是一款功能强大的代码编辑器，支持多种编程语言，并拥有丰富的插件生态系统。我主要用它来编写和调试代码，以及进行项目管理。
  • Notepad++：这款轻量级文本编辑器简洁高效，适合快速编辑文本和代码文件。我常用它来处理配置文件和其他小型项目。

Markdown 编辑

  • Typora：Typora 是一款优雅的 Markdown 编辑器，提供实时预览和多种导出格式的支持。它简洁的界面和流畅的操作体验，让我能够专注于写作和文档编辑工作。

图像优化与资源

  • Squoosh：Squoosh 是一款图像压缩工具，能够在保证质量的前提下极大地减少图像文件的大小。我使用它来优化博客中使用的图片，以确保页面加载速度快。
  • Unsplash：Unsplash 提供了大量高质量的免费图片资源，为我的博客内容增添了视觉吸引力。无论是文章封面还是插图，Unsplash 都是我的首选图库。

域名与安全管理

  • NameSilo：我使用 NameSilo 购买和管理博客的域名。它的易用性和价格优势使得我能够轻松地管理博客的域名。
  • Cloudflare：为了提升网站的安全性和访问速度，我将博客的流量通过 Cloudflare 进行加速和保护。它提供的 CDN 和 DDoS 防护功能，使得我的网站能够应对各种网络挑战，并为用户提供快速稳定的访问体验。

托管与部署

  • GitHub Pages：通过 GitHub Pages 服务，我可以将 Hugo 生成的静态网站直接发布到互联网上，并进行自动化的持续集成和部署。
  • Vercel：Vercel 是我用来部署 Waline 评论系统的平台。它确保了系统的快速响应，并且提升了整体用户体验。

评论系统与邮件管理

  • Waline：Waline 是一款轻量级的评论系统，提供高效、安全且易于使用的评论功能。我使用它来处理博客上的评论和反馈，让与读者的互动更加方便。
  • Resend：我使用 Resend 来管理博客的邮件发送功能，确保联系表单和其他邮件通知的顺利发送。

通过这些工具的协同工作，我能够高效地管理和发布博客内容，希望您在这里能够找到对您有用的资源和灵感。感谢您的访问！
